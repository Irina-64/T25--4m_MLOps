# Инструкция по запуску фронтенда

## Быстрый старт

### Локальный запуск

1. Убедитесь, что установлены все зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите API сервер:
   ```bash
   python3 -m uvicorn src.api:app --host 0.0.0.0 --port 8080
   ```
   
   Или если uvicorn установлен в PATH:
   ```bash
   uvicorn src.api:app --host 0.0.0.0 --port 8080
   ```

3. Откройте браузер и перейдите по адресу:
   ```
   http://localhost:8080
   ```

### Запуск через Docker

1. Соберите и запустите контейнер:
   ```bash
   docker-compose -f docker-compose.api.yaml up --build
   ```

2. Откройте браузер и перейдите по адресу:
   ```
   http://localhost:8080
   ```

## Использование

1. Введите текст в текстовое поле
2. При необходимости настройте параметры:
   - **Максимальная длина**: максимальная длина выходного текста (16-256)
   - **Количество beams**: параметр beam search для генерации (1-10)
3. Нажмите кнопку "Детоксифицировать"
4. Результат появится ниже с сравнением исходного и детоксифицированного текста

## Структура фронтенда

- `frontend/index.html` - основной HTML файл
- `frontend/style.css` - стили интерфейса
- `frontend/script.js` - JavaScript для взаимодействия с API

## API эндпоинты

- `GET /` - главная страница с фронтендом
- `GET /health` - проверка здоровья API
- `POST /predict` - детоксификация текста
- `GET /metrics` - метрики Prometheus
